Introduction

Purpose

The aim of this system requirements specification document is to provide an in-depth description of the tool Meyer's Inventory System. This document will explain the features and the purpose of this software.

Product Scope

Meyer's Inventory System is a website that allows admins to manage (delete / update / add) inventory products, brands, categories. This tool allows admins to visualize these data in table form.

Overall Description

Product Perspective

Meyer's Inventory System is a website that offers data about products, brands, category, orders. Admins can manage these data. These data can be visualized as tables.

Product Functions

  • Dashboard page

    • Provides dashboard cards for every table of data (categories, brands, products, orders).
    • Every dashboard cards contains a title and the number of table rows corresponding to that title.

  • Login page

    • Allow admins only to log in.

  • Categories page

    • Allow admins to visualize data in table form.
    • Allow admins to update, add, delete a category.
    • Every row of category table has a title, added by, added on and actions(update / delete).

  • Brands page

    • Allow admins to visualize data in table form.
    • Allow admins to update, add, delete a brand.
    • Every row of brands table has a title, added by, added on and actions(update / delete).

  • Products page

    • Allow admins to visualize data in table form.
    • Allow admins to update, add, delete a product.
    • Every row of products table has an id, name, brand, category, color, size, genre, quantuty and actions(update / delete).

  • Orders page

    • Allow admins to visualize orders in table form.
    • Allow admins to update, add, delete a product.
    • Every order has a bill no, customer name, customer adress, id product, quantity, total amount (in euros), actions

User Classes and Characteristics

  • The website is forbidden for regular users.
  • Admins must be logged in to be able to manage the inventory.

Operating Environment

This website can be hosted on any kind of operating system that can run PHP, Laravel: Unix-like operating systems, Windows and macOS.

Design and Implementation Constraints

This software is developed using PHP, Laravel, HTML and CSS.

Assumptions and Dependencies

The site depends on PHP 8 and a web server such as Apache or nginx.
